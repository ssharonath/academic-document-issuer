# Academic Certificate System

## Start the system

Double-click:

`START-CERTIFICATE-SYSTEM.command`

Then open:

`http://127.0.0.1:4173/?v=progress`

## Main screens

- Dashboard: progress, audit, issued certificate counts.
- Template Designer: move fields, add QR, seals, text, tables, logos, signatures, backgrounds.
- Bulk Batch Process: upload CSV from Excel and generate certificates in batches.
- Students & Masters: manage student and academic master data.
- Transcript Builder: module table, GPA, credits and classifications.
- Approvals & Locking: authority-level approval flow.
- Verification: search certificate number / QR verification simulation.
- Audit & Reports: activity trace and exports.

## Current version
