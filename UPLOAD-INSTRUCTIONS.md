# Academic Document Issuer - Final Upload Instructions

This folder is the clean final upload folder.

## Upload To GitHub

1. Open:
   https://github.com/ssharonath/academic-document-issuer

2. Click **Add file**.

3. Click **Upload files**.

4. Drag every file from this folder into GitHub:

   - `index.html`
   - `app.js`
   - `styles.css`
   - `config.js`
   - `package.json`
   - `vercel.json`
   - `README.md`
   - `UPLOAD-INSTRUCTIONS.md`

5. Scroll down.

6. Click **Commit changes**.

## Vercel

1. Open the Vercel project named `academic-document-issuer`.

2. Wait until the new deployment says **Ready**.

3. Open:
   https://academic-document-issuer.vercel.app?v=final-upload

## Login

Use the email and password created in Supabase Authentication.

The Super Admin role is controlled by the `staff_profiles` table in Supabase.

## Do Not Upload These

Do not upload files that show only:

```text
404: Not Found
```

Those files are broken and will make Vercel fail.
